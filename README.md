# wca-333fm-stats.jl

WCA (World Cube Association) 三阶最少步 (333fm) 统计数据分析和可视化工具。

## 功能特性

- 📊 **统计分析**：计算各种统计指标（最佳成绩、平均成绩、单次成绩等）
- 🏆 **排名系统**：所有数值列都有对应的全球排名和国家排名，三列绑定显示
- 📈 **滚动平均**：MO3/5/12/50/100, AO5/12/50/100
- 🎯 **奖牌统计**：金/银/铜牌统计
- 🌐 **Web 前端**：美观的数据可视化界面
- 📁 **多文件支持**：可在三个数据文件之间切换查看
- 🔧 **列选择器**：自定义显示哪些列，按类别组织
- 🔄 **排名显示开关**：全局控制是否显示排名信息
- 📊 **智能排序**：默认按排名列排序，确保结果准确
- 🔢 **智能格式化**：次数/个数和单次/最佳的非计算值自动显示为整数
- ✅ **数据完整性检查**：自动检查并提示缺失的排名列
- 💾 **数据持久化**：自动保存用户的列选择、文件选择和显示偏好

## 使用方法

### 1. 生成统计数据

运行 Julia 脚本生成 CSV 文件：

```bash
julia -e 'include("WCAStats.jl"); WCAStats.__init__()' source <WCA_export_file.zip>
```

生成的 CSV 文件：
- `results.csv` - 完整统计数据
- `results.top100.csv` - 全球 Top 100
- `results.china.top30.csv` - 中国 Top 30

### 2. 查看前端页面

1. 确保已生成至少一个 CSV 文件（`results.csv`、`results.top100.csv` 或 `results.china.top30.csv`）
2. 在浏览器中打开 `index.html` 文件
   - 可以直接双击打开（某些浏览器可能限制本地文件访问）
   - 或者使用本地服务器（推荐）：
     ```bash
     # Python 3
     python -m http.server 8000
     
     # 或使用 Node.js
     npx http-server
     ```
3. 访问 `http://localhost:8000` 查看页面

## 前端功能

### 数据展示
- 📊 **统计摘要**：总选手数、平均最佳成绩、平均成绩、总比赛次数
- 📋 **数据表格**：详细的选手统计数据，支持自定义列显示
- 🏅 **奖牌展示**：金/银/铜牌数量
- 📁 **多文件切换**：可在三个数据文件之间切换
  - 中国 Top 30 (`results.china.top30.csv`)
  - 全球 Top 100 (`results.top100.csv`)
  - 完整数据 (`results.csv`)

### 交互功能
- 🔍 **搜索**：按姓名搜索选手
- 📊 **排序**：
  - 点击表头或使用下拉菜单排序
  - **默认按排名列排序**（即使排名未显示），确保排序准确
  - 有排名列的字段优先使用排名列排序
  - 支持所有列排序
- 📄 **分页**：每页显示 30 条记录
- 🔄 **排名显示开关**：
  - 全局控制是否显示排名信息
  - 关闭时：只显示原始数据值
  - 打开时：显示 `原始值 (全球排名/国家排名)` 格式
  - 设置自动保存
- 🔧 **列选择器**：
  - 按类别组织列（基本信息、比赛统计、最佳成绩、平均成绩、单次成绩、滚动平均、奖牌等）
  - **列绑定系统**：原始数据列、全球排名列、国家排名列自动绑定为一组
  - 选择基础列时，排名信息会根据开关自动显示/隐藏
  - 每个分组提供全选/全不选按钮
  - 支持显示/隐藏任意列
  - 每个文件的列选择设置独立保存
- ✅ **数据完整性检查**：
  - 加载数据时自动检查有排名列的字段是否三列齐全
  - 缺少排名列时会显示详细错误信息
- 💾 **数据持久化**：
  - 自动保存文件选择
  - 自动保存每个文件的列选择设置
  - 自动保存排名显示开关状态
  - 下次打开页面时自动恢复所有设置
- 📱 **响应式设计**：支持移动设备

### 列分类
- **基本信息**：姓名、ID、国家、性别（无排名列）
- **比赛统计**：比赛次数、轮次、机会数、尝试次数（整数显示）
- **最佳成绩**：最佳成绩及其各种统计指标
  - 整数：最佳、最佳最大值、最佳众数、最佳连续范围等
  - 小数：最佳平均、最佳标准差、最佳中位数等计算值
- **平均成绩**：平均成绩及其各种统计指标
  - 整数：平均次数、平均唯一值数、平均众数次数、平均连续次数等
  - 小数：平均、平均最大值、平均均值、平均标准差等计算值
- **单次成绩**：单次成绩及其各种统计指标
  - 整数：成功次数、单次最小值/最大值、单次众数、单次连续范围等
  - 小数：单次平均、单次标准差、单次中位数等计算值
- **滚动平均 - MO**：MO3/5/12/50/100 的最后和最佳值（小数）
- **滚动平均 - AO**：AO5/12/50/100 的最后和最佳值（小数）
- **奖牌**：金/银/铜牌数量（整数）
- **其他**：平均项统计（整数）

### 列绑定系统
所有数值列都自动绑定三列：
- **原始数据列**：如 `best`、`average`、`solved_mean` 等
- **全球排名列**：如 `best_rank`、`average_rank` 等
- **国家排名列**：如 `best_nr`、`average_nr` 等

在列选择器中只显示基础列，排名信息通过全局开关控制显示。

## 数据字段说明

### 基本统计
- `best` - 最佳成绩（整数）
- `average` - 平均成绩（小数）
- `solved_mean` - 单次平均成绩（小数）
- `competitions` - 比赛次数（整数）
- `rounds` - 轮次（整数）
- `solved_count` - 成功次数（整数）
- `chances` - 机会数（整数）
- `attempts` - 尝试次数（整数）

### 排名系统
**所有数值列都有对应的排名列，三列自动绑定**：
- **原始数据列**：如 `best`、`average`、`solved_mean` 等
- **全球排名列**：`*_rank` - 全球排名（Global Rank）
- **国家排名列**：`*_nr` - 国家排名（National Rank）

例如：
- `best` → `best_rank`（全球排名）、`best_nr`（国家排名）
- `average` → `average_rank`（全球排名）、`average_nr`（国家排名）
- `solved_mean` → `solved_mean_rank`（全球排名）、`solved_mean_nr`（国家排名）
- `solved_mo3_last` → `solved_mo3_last_rank`（全球排名）、`solved_mo3_last_nr`（国家排名）

**显示格式**：
- 关闭排名显示：只显示原始值
- 打开排名显示：`原始值 (全球排名/国家排名)`

### 统计指标
每个成绩类型都包含多种统计指标：

**整数类型**（次数/个数/非计算值）：
- `*_count` - 次数
- `*_nunique` - 唯一值数量
- `*_mode_count` - 众数次数
- `*_consecutive` - 连续次数
- `*_min` / `*_max` - 最小值/最大值（单次/最佳的实际值）
- `*_mode` - 众数（实际出现的值）
- `*_consecutive_start` / `*_consecutive_end` - 连续范围（最佳/单次的实际值）

**小数类型**（计算值）：
- `*_mean` - 平均值
- `*_std` - 标准差
- `*_avg` - 去极值平均值
- `*_median` - 中位数
- `*_consecutive_start` / `*_consecutive_end` - 连续范围（平均值的计算值）

### 奖牌
- `gold` - 金牌数（整数）
- `silver` - 银牌数（整数）
- `bronze` - 铜牌数（整数）

### 滚动平均
- `solved_mo3/5/12/50/100_last` - Mean of 3/5/12/50/100 最后值（小数）
- `solved_mo3/5/12/50/100_best` - Mean of 3/5/12/50/100 最佳值（小数）
- `solved_ao5/12/50/100_last` - Average of 5/12/50/100 最后值（小数）
- `solved_ao5/12/50/100_best` - Average of 5/12/50/100 最佳值（小数）

## 使用技巧

### 排名显示控制
- 使用"显示排名"复选框控制是否显示排名信息
- 关闭时：表格更简洁，只显示原始数据
- 打开时：显示完整信息，格式为 `原始值 (全球排名/国家排名)`
- 设置会自动保存，下次打开页面时恢复

### 列选择器使用
1. 点击"选择列..."按钮打开列选择面板
2. 按类别浏览和选择要显示的基础列（排名列会自动绑定）
3. 使用分组标题旁的"全选"/"全不选"按钮快速选择整个类别
4. 点击"应用"按钮保存选择并更新表格
5. 每个文件的列选择设置会独立保存

### 文件切换
- 在控制面板顶部的"数据文件"下拉菜单中选择要查看的文件
- 页面会自动加载对应的 CSV 文件
- 列选择设置会按文件分别保存和恢复
- 如果数据不完整（缺少排名列），会显示错误提示

### 排序功能
- **默认按排名排序**：即使排名未显示，排序也会使用排名列，确保结果准确
- 点击表头可以快速排序（会切换升序/降序）
- 使用排序下拉菜单选择排序字段和方式
- 有排名列的字段会自动使用排名列排序

### 搜索功能
- 在搜索框输入姓名可以实时过滤数据
- 支持中文和英文姓名搜索

### 数据格式
- **整数列**：次数、个数、单次/最佳的非计算值自动显示为整数
- **小数列**：平均值、标准差、中位数等计算值显示为两位小数
- 排名信息始终显示为整数

## 技术栈

- **后端**: Julia + DataFrames + StatsBase + RollingFunctions
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **数据格式**: CSV
- **存储**: LocalStorage（保存用户偏好设置）
